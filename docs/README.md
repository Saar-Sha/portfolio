# Saar Sharir — Portfolio Website

A modern personal portfolio website. No build tools, no frameworks — fast, maintainable static files.

## Quick Start

1. Download the project folder
2. Open `index.html` in any browser — works locally
3. Deploy publicly via GitHub Pages, Netlify, or Vercel (see DEPLOYMENT.md)

## Pages

| Page | File | Purpose |
|------|------|---------|
| Homepage | index.html | Hero, about, featured projects, skills, experience, contact |
| Projects | pages/projects.html | Full project browser with tabs and filters |
| About | pages/about.html | Detailed profile, philosophy, approach |

## Key Features

- Dark/Light mode — persisted, respects OS preference
- Project filtering by technology tag
- Project detail modals with full case study view
- Scroll-triggered animations and skill bars
- Contact form — plug in Formspree/EmailJS
- Analytics — localStorage tracking + Plausible/Umami hooks
- Fully responsive

## Updating Personal Info

Search these placeholders in HTML files and replace:

| Placeholder | Replace With |
|-------------|-------------|
| saar@example.com | Your real email |
| https://github.com/Saar-Sha | Your GitHub URL |
| www.linkedin.com/in/saar-sharir | Your LinkedIn URL |
| ./assets/saar-sharir-cv.pdf | Your resume PDF path |

See ADD_NEW_PROJECT.md, DEPLOYMENT.md, ANALYTICS.md for detailed guides.
